# FacMod

[![CI](https://github.com/ajilenakh/FacMod/actions/workflows/ci.yml/badge.svg)](https://github.com/ajilenakh/FacMod/actions/workflows/ci.yml)

One-click Factorio mod downloads from [mods.factorio.com](https://mods.factorio.com). No account required.

Backend by [re146.dev](https://re146.dev/).

## Install
Download [Firefox addon](https://addons.mozilla.org/en-US/firefox/addon/facmod-factorio-mod-downloader) here.
